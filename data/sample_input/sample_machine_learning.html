Градиентный спуск и обучение модели

Машинное обучение изучает методы, которые позволяют модели находить закономерности в данных. В задачах classification и regression модель обучается на тренировочном наборе, а качество проверяется на тестовой выборке.

Простейшая формула ошибки может быть записана как y = f(x). Для обучения используется градиентный спуск, регуляризация и анализ признаков. 

def train(model, dataset):
    return model.fit(dataset)
